# wiziqiGame

## 介绍

wiziqiGame是一款基于tcp的在线五子棋游戏，可以实现玩家对战，人机对战，在线对战。

## 软件架构

C/S架构，客户端采用qt开发GUI界面，服务端在linux平台开发
